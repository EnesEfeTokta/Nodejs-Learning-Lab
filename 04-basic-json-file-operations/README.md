# 04 Basic JSON File Operations
This directory contains examples and exercises related to basic JSON file operations in JavaScript. You will learn how to read from and write to JSON files using the built-in `json` module.

## Available Andpoints
- `readJsonFile(filePath)` - Reads a JSON file from the specified `filePath` and returns the parsed JavaScript object.
- `writeJsonFile(filePath, data)` - Writes the provided JavaScript object `data` to a JSON file at the specified `filePath`.
