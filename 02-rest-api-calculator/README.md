# 02 - Rest API Calculator

This mini-project builds a simple REST API calculator using Node.js and Express.

## Concepts covered
- Query parameters
- Basic arithmetic operations via API endpoints
- Error handling for invalid inputs
This is a learning-focused project, not production-ready.

## Available Endpoints
- `GET /api/sum?a=NUMBER&b=NUMBER` - Returns the sum of `a` and `b`.
- `GET /api/subtract?a=NUMBER&b=NUMBER` - Returns the difference of `a` and `b`.
- `GET /api/multiply?a=NUMBER&b=NUMBER` - Returns the product of `a` and `b`.
- `GET /api/divide?a=NUMBER&b=NUMBER` - Returns the quotient of `a` divided by `b`.
